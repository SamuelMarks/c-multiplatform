# LibCMPC

Welcome to the official documentation for **LibCMPC**, the C89 cross-platform application framework.
LibCMPC provides a robust, zero-dependency, highly scalable UI and system abstraction layer targeting desktop, mobile, and web.

## Getting Started

- \subpage tour "Tour the Widgets" - See the visual design system across all platforms.
- \subpage philosophy "Theory & Philosophy" - Read about the architectural decisions driving LibCMPC.
- [API Documentation](annotated.html) - Browse the C89 API references.

## Features

- **Strict C89:** Compiles everywhere, from MSVC to GCC to Emscripten.
- **Dependency-free:** Implements its own rendering, layout, and data structures.
- **Cross-Platform:** Write once, run on Windows, macOS, Linux, Android, iOS, and the Web.
- **Scalable UI:** Fully implemented Material Design 3 (M3) widget set.
